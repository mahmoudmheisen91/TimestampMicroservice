Timestamp Microservice is a NodeJS API that allow the client to request date using a route parameter, server respond with a JSON object containing UNIX and UTC Format of the date, done for FreeCodeCamp APIs and Microservices Course, Project 1


Example Usage:

URL OUTPUT
GET [project url]/api/timestamp/2020-1-1 {"unix":1577836800000,"utc":"Wed, 01 Jan 2020 00:00:00 GMT"}
GET [project url]/api/timestamp/1450137600 {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
GET [project url]/api/timestamp/ {"unix":1584152780307,"utc":"Sat, 14 Mar 2020 02:26:20 GMT"}
GET [project url]/api/timestamp/22-2-2 {"unix": null, "utc" : "Invalid Date" }
